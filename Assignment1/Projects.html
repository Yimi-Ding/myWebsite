Tool Rental Management System

October 2024, Database System Course, Algonquin College, Ottawa, ON

Project Overview

The Tool Rental Management System is a database application designed to simplify the management of tool rentals. Built using Microsoft Access and SQL Server Management Studio (SSMS), this system allows rental companies to keep track of tool categories, individual tools, rental transactions, maintenance records, and staff involvement. It provides an organized, efficient way to handle tool availability, rental history, and maintenance needs.

Key Features

  • Tool Categories and Inventory Management: Organizes tools into categories, enabling users to easily find and manage tool types.
  • Rental Transactions: Tracks detailed information about each rental, including the renter, rental period, fees, and the tools rented.
  • Maintenance Records: Logs maintenance activities for each tool, ensuring tools are in safe, working condition before being rented out.
  • Staff Management: Records staff members involved in rental and maintenance processes, promoting accountability and efficient tracking.

Use Cases

This system is ideal for companies renting out tools and equipment, as it improves organization, enhances tracking of rentals and returns, and ensures timely maintenance. By streamlining these processes, the Tool Rental Management System helps maintain an efficient workflow and ensures high customer satisfaction through better service management.

Baggage Checking Simulation Program

July 2024, Computer Programming Course, Algonquin College, Ottawa, ON

Project Overview

The Baggage Checking Simulation Program is a Java-based application where users input baggage dimensions (length, width, height) and weight. The program checks these entries against airline standards, helping airlines manage baggage more efficiently by reducing issues with non-compliant luggage.

Key Features

  • Baggage Input: Users can enter details for multiple pieces of baggage, with each entry creating a Baggage object.
  • Compliance Check: The program validates each baggage entry against airline size and weight limits, providing immediate feedback on compliance.
  • Detailed Output: After all entries, the program displays details and compliance results for each piece of baggage.

Use Cases

This simulation is useful for training airline staff or as part of a larger baggage management system, enhancing the efficiency of baggage checks for ground personnel.
